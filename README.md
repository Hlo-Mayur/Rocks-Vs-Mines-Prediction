# Rocks vs Mines Prediction ⛏️

This project uses machine learning to classify sonar signals as either **rock** or **mine**.

## Project Structure
Rocks-Vs-Mines-Prediction/
│── data/ # Dataset files
│── notebooks/ # Jupyter notebooks
│ └── rocks_vs_mines.ipynb
│── README.md
│── requirements.txt

## Technologies Used
- Python
- NumPy
- Pandas
- Scikit-learn
- Matplotlib

## Dataset
The dataset contains sonar signal readings used to distinguish rocks from mines.

## Objective
To build and evaluate a supervised ML model for binary classification.
